# Subscription Intelligence Dashboard
This project analyzes user subscription data to identify spending patterns and detect wasted expenses.

# Problem
Users subscribe to multiple services but often do not track usage, leading to unnecessary spending.

# 💡 Solution

This dashboard provides:

Total subscription spending
Category-wise breakdown
Waste detection using usage patterns
Estimated monthly waste

# Tech Stack
Power BI
Excel
DAX

# Features
KPI cards
Interactive charts
Waste detection logic
Data-driven insights
