# 📊 Subscription Intelligence Dashboard

A data-driven Power BI dashboard to analyze subscription usage patterns and identify wasted spending.

---

## 🎯 Problem Statement
Users subscribe to multiple digital services but often fail to track their actual usage.  
This leads to **unnecessary monthly expenses** and inefficient financial decisions.

---

## 💡 Solution
This project provides a Power BI dashboard that:

- Tracks total subscription spending  
- Breaks down spending by category  
- Identifies inactive or low-usage subscriptions  
- Calculates estimated monthly waste  

---

## 📊 Key Insights

- 💰 **Total Monthly Spend:** ₹8000  
- ⚠️ **Wasted Amount:** ₹2350  
- 📉 **~28% of spending is wasted**

---

## 🚀 Features

- 📌 KPI Cards for quick insights  
- 📊 Category-wise spending analysis  
- ⚠️ Waste detection using usage patterns  
- 🧠 DAX-based calculated columns & measures  
- 📉 Visual insights for decision-making  

---

## ⚙️ Tech Stack

- **Power BI**
- **Microsoft Excel**
- **DAX (Data Analysis Expressions)**

---

## 🧠 Methodology

- Created structured dataset of user subscriptions  
- Built data model in Power BI  
- Developed DAX logic to classify subscriptions as:
  - **Active**
  - **Waste** (based on usage & inactivity)
- Designed interactive dashboard for insights
